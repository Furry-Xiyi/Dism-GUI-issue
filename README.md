## 简体中文说明

### 🧹 空间回收
提供系统级的磁盘清理能力，支持扫描临时文件、更新残留、日志文件、缓存等占用空间的项目，并允许用户按需清理，释放磁盘空间、提升系统可用性。

### 🚀 启动项管理
列出所有开机启动项，包括注册表启动项、计划任务、自启动服务等，支持启用、禁用、删除，帮助用户加快系统启动速度、减少后台占用。

### 📦 应用管理
支持查看、卸载系统应用（Appx/MSIX）与传统 Win32 程序，提供包信息、安装路径、依赖项等详细数据，方便用户管理系统中的所有软件组件。

### 🧰 工具箱
集成多种系统维护工具，包括系统备份、系统还原、账户管理、引导修复、ESD/WIM 转换、镜像处理、Hosts 编辑等，提供一站式系统维护能力。

### ⚙️ 系统优化
提供常用的系统优化选项，如服务管理、计划任务管理、网络优化、视觉效果调整等，帮助用户提升系统性能与响应速度。

### 🖥️ 驱动管理
支持查看系统驱动、导出驱动列表、安装或删除驱动包，帮助用户维护系统驱动环境，解决驱动冲突或缺失问题。

### 📋 程序和功能
支持查看、卸载传统桌面程序，并显示安装时间、版本号、发布者等详细信息。支持安装、卸载现代可选功能。

### 🧩 可选功能
列出 Windows 可选功能（如 .NET Framework、Hyper-V、Telnet 等），支持启用、禁用、搜索、筛选，并提供实时状态显示，方便用户管理系统组件。

### 🔄 更新管理
提供 Windows 更新管理能力，包括查看更新历史、卸载更新、隐藏更新、安装本地更新包等，帮助用户更灵活地控制系统更新行为。

### 📁 文件关联
允许用户查看和修改文件类型与默认应用的关联关系，支持重置、切换、恢复默认设置，提升文件打开体验。

### 📝 预应答
支持创建、编辑 Windows 无人值守安装（Unattend.xml）配置文件，提供图形化编辑界面，帮助用户快速生成自动化部署所需的应答文件。

## English Description

### 🧹 Space Cleanup
Provides system-level disk cleanup capabilities, supporting scanning of temporary files, update leftovers, log files, caches, and other space-consuming items. Users can clean up as needed to free disk space and improve system usability.

### 🚀 Startup Management
Lists all startup items, including registry entries, scheduled tasks, and auto-start services. Supports enabling, disabling, and deleting to speed up system boot and reduce background usage.

### 📦 Application Management
Supports viewing and uninstalling system apps (Appx/MSIX) and traditional Win32 programs. Provides detailed package information, installation paths, and dependencies for easier software management.

### 🧰 Toolbox
Integrates multiple system maintenance tools, including system backup, system restore, account management, boot repair, ESD/WIM conversion, image handling, and Hosts editing, offering one-stop system maintenance.

### ⚙️ System Optimization
Provides common optimization options such as service management, task scheduling, network tuning, and visual effects adjustment to improve performance and responsiveness.

### 🖥️ Driver Management
Supports viewing system drivers, exporting driver lists, installing or removing driver packages to maintain driver environment and resolve conflicts or missing drivers.

### 📋 Programs and Features
Supports viewing and uninstalling traditional desktop programs, showing installation time, version, and publisher. Also supports installing and uninstalling modern optional features.

### 🧩 Optional Features
Lists Windows optional features (e.g., .NET Framework, Hyper-V, Telnet), supports enabling/disabling, searching, filtering, and provides real-time status display for easier component management.

### 🔄 Update Management
Provides Windows Update management capabilities, including viewing update history, uninstalling updates, hiding updates, and installing local update packages for flexible control.

### 📁 File Associations
Allows users to view and modify file type associations with default apps, supporting reset, switching, and restoring defaults to improve file opening experience.

### 📝 Unattended Setup
Supports creating and editing Windows unattended installation (Unattend.xml) configuration files with a graphical editor, helping users quickly generate automated deployment answer files.
